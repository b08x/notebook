====== qmidiarp ======
Created Monday 27 September 2021

0..9 : Note indices
   + : One octave up
   - : One octave down
   t : One semitone up
   g : One semitone down
   = : Reset to standard octave and semitone
   > : Double tempo
   < : Half tempo
   . : Reset to standard tempo
 ( ) : group chord notes   
   / : Volume up by 20%
   \ : Volume down by 20%
   d : Double length
   h : Half length
   p : Pause
